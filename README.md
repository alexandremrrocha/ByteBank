
# Byte Bank

Tela de transferencia bancaria com a demonstração do extrato a cada transferencia executada

## Stack utilizada

- Angular

## Aprendizados

- TypeScript
- Angular
- SCSS

## Rodando localmente

- Clone o projeto

- Entre no diretório do projeto

- Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  ng serve
```
