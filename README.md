
# Byte Bank

Projeto para o aprendizado do framework do Angular, projeto em andamento

## Stack utilizada

- Angular

## Aprendizados

- TypeScript
- Angular
- SCSS

## Rodando localmente

- Clone o projeto

- Entre no diretório do projeto

- Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  ng serve
```
